## Installation

1. Clone the repository:
   ```git clone https://github.com/your-username/currency-dashboard.git ```

2. Navigate to the project directory: 
   ```cd currency-dashboard ```

3. Install the dependencies: 
   ``` npm install ```

4. Start the development server: 
   ```npm run dev ```
